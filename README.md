Deskripsi Singkat
---
Project ini merupakan aplikasi web sederhana untuk manajemen data produk yang terdiri dari tiga halaman utama, yaitu Login Page, Dashboard, dan Products Page.  
Aplikasi dibuat menggunakan HTML, CSS, dan JavaScript murni tanpa framework, sehingga mudah dipahami dan dipelajari.  

1. Login Page
Halaman login digunakan untuk autentikasi pengguna sebelum masuk ke sistem.  
Pengguna harus memasukkan email dan password yang valid (password: 24090070).  
Jika input salah atau kosong, sistem menampilkan pesan error.  
Tampilan login dirancang modern dengan ilustrasi gambar dan warna bertema biru.  

2. Dashboard Page  
Setelah login berhasil, pengguna diarahkan ke halaman dashboard.  
Dashboard menampilkan ringkasan data berupa:  
Total produk  
Total penjualan  
Total pendapatan  
Data ditampilkan dalam bentuk card dengan desain modern.  
Dashboard juga menyediakan tombol untuk menuju halaman Products.  

3. Products Page  
Halaman ini berfungsi untuk menampilkan daftar produk dalam bentuk tabel.  
Setiap produk memiliki data:
Nama produk  
Harga  
Stok
Halaman ini menyediakan fitur:  
Edit produk (dummy alert)  
Delete produk dengan modal konfirmasi  
Tombol "Add Product" (belum berfungsi / dummy)  
Tabel produk dirender secara dinamis menggunakan JavaScript.  

5. Fitur Responsif (Mobile Friendly)  
Proyek ini dilengkapi hamburger menu untuk tampilan mobile.  
Sidebar dapat dibuka-tutup dengan animasi melalui JavaScript dan overlay background.
